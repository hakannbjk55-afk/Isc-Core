Three-Layer Hash Model
meta_hash
=
SHA256(canonical(ci_report.json))
content_hash
=
SHA256(canonical(content_manifest))
pack_hash
=
SHA256(meta_hash || content_hash)
anchor (optional)
=
pack_hash → Base L2 (--verify-anchor)
isc_verify — Verification Algorithm
1
Extract + Validate Structure
tar extract → path traversal check → required files present
2
Content Integrity
for each file in manifest: SHA256(file) == manifest_hash → content_hash
3
Pack Identity
SHA256(meta_hash || content_hash) == stored pack_hash
4
Governance Verification
signer in allowed_signers AND NOT in revocation_record
5
Signature Verification
ed25519_verify(sig, payload, public_key) → 3 signatures
6
Lineage Verification (V2)
for parent in parents: parent.outputs ⊆ child.inputs
7
Anchor Verification (optional)
--verify-anchor: pack_hash exists on Base L2
